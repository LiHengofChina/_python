	深度学习中，让机器进行学习，它用的网络结构是：深度神经网络结构。

//============================================================ 神经网络
//============================================================ 神经网络

	"神经网络" 要去预测，要去学习，它的核心套路是什么？
			求 "损失函数的极小值"

		//在深度学习中， "更新模型参数"，依然是 "梯度下降法"
		//但是没有办法直接梯度下降了，

		//因为 "线性回归" 可以直接求导。
		// "深度神经网络" 里面没有办法直接求导。
				 /**
					因为它的参数数据是一层一层的，求一层，反向传播误差，再求
					链式法则反向逐层求梯度。
				 */

		// 那就需要想方设法来求导，计算梯度，再进行 "梯度下降"。

//============================================================ 深度神经网络的派生
//============================================================ 深度神经网络的派生

	（1）CNN ：卷积神经网络
				//卷积 + 池化，擅于抓局部特征	——>	图像、OCR。
	（2）GAN ：生成对抗网络
				//生成器 + 判别器对抗			——>	图像生成

	（3）RNN ：循环神经网络
				//循环结构，擅于处理序列	 	——>	文本、时序

	（4）Transformer//序列 + 生成，当今 LLM 主流
				广义上：Transformer 也是深度神经网络的一种架构（多层 Attention + 全连接）
				狭义上：它不是从 CNN/RNN/GAN 派生出来的，而是另起炉灶，用 Attention 取代 RNN 处理序列
