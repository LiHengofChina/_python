Ollama 是美国团队做的开源项目（公司 Ollama），大约在 2023 年公开并很快流行起来。

//======================================================= Ollama
//======================================================= Ollama

Ollama 是一个在 "本机或服务器" 上运行大语言模型的工具，
	 可以理解为：帮你把各种开源模型（如 Llama、Qwen、Mistral 等）
	 			下载、管理、跑起来，
	 并提供 API 服务。

//======================================================= 它主要做什么：
//======================================================= 它主要做什么：

（1）安装后，用命令行拉取模型、启动对话；
（2）在后台跑一个本地推理服务，默认常见端口是 11434；
（3）提供 HTTP API，其中常带 兼容 OpenAI 的接口（例如 /v1/chat/completions），所以很多程序里填 OPENAI_BASE_URL=http://本机:11434/v1 就能当作「本地 OpenAI」来用。

//======================================================= 和云上大模型的区别：
//======================================================= 和云上大模型的区别：

（1）数据、推理可以完全在自己机器/内网里，不经过公网（适合内网、涉密场景）；
（2）性能取决于你的 CPU/GPU 和内存，大模型要跑得动需要硬件够。
一句话： Ollama = 本地跑开源 LLM 的轻量运行时 + 命令行/REST API，
方便集成到你自己的系统里。
