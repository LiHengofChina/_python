//=============================== Prompt 是什么？

你给大模型的输入指令/问题，模型根据它 "生成回答"。

Prompt = 你对 AI 说的话（含角色、任务、格式要求）


//========== 例子：
//========== 例子：

（1）「把下面日志总结成 3 条」→ Prompt
（2）「你是运维助手，分析这段报错」→ Prompt（还带 角色设定）

//===============================
//===============================

Prompt = 你怎么跟大模型说话；写得好，回答才准。
