
//================================== Token
//================================== Token

Transformer 输入的是数字序列，不能直接吃汉字/英文。
所以要先把文本切成 Token，再送进模型。

//================================== Embedding
//================================== Embedding

每个 Token 要变成 向量 才能进 Transformer 算 Attention。
这一步就是 Embedding（查表或矩阵映射）。

//================================== 上下文窗口
//================================== 上下文窗口

Transformer 一次 Self-Attention 要算 所有 Token 之间谁关注谁。
Token 越多，计算量 平方级增长（n 个词要算 n×n 的关系）。
所以模型会限制 一次最多处理多少个 Token → 这就是 上下文窗口（如 4K、128K）。

Token 是输入单位，Embedding 是进 Transformer 的格式，上下文窗口是 Transformer 一次能处理的 Token 上限。
