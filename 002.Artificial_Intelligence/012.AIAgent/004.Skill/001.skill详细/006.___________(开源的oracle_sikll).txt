//=========================================================== 使用示例
//=========================================================== 使用示例

以 https://github.com/oracle/skills 的 db 域为例

（1）下载 Skill 到本地
    skills/oracle/db/

（2）编写 Agent，加载 Skill
    workspace/day_03_oracle_skill_agent/

（3）配置
    cd workspace/day_03_oracle_skill_agent
    copy config.example.yml config.local.yml
    填 ssh（已有 Oracle 再开 oracle.enabled=true）

（4）安装依赖
    pip install -r requirements.txt

（5）运行
	python main.py "查一下 lg_user 有哪些表，并看当前实例版本"

	python main.py "Oracle 有一条 SQL 很慢，帮我定位"
