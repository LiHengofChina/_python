Skill：磁盘告警排查
├─ 触发条件：用户说磁盘告警 / 监控触发
├─ 指令：先查 df，再 du，再查手册，再给建议
├─ 使用的 Tool：ssh_exec, rag_search
├─ 步骤清单：1…2…3…（类似你 Cursor 里的 skill.md）
└─ 验收：是否给出根因 + 可操作方案

Cursor 的 Skill（.cursor/skills/）就是这个思路在 IDE 里的实现。
Cursor 的 Skill 主要是 Markdown 指令文件（SKILL.md），给 Agent 看的工作流说明，不是用 C++ 写的业务逻辑。
