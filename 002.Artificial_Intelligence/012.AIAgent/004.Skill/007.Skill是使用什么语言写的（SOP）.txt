//================================================== Skill 用什么写？

Skill 不是 Python 代码，是 带元数据的 SOP 包；description 决定以后会不会被选中。


//==================================================
//==================================================

SOP = Standard Operating Procedure，标准作业流程。

就是一套固定步骤：遇到某类问题，先做什么、再做什么、最后怎么验收。

Skill 里的 SOP 例子：

1. 先 ssh 连机器
2. 再 df -h 看分区
3. 再 du 查大目录
4. 查手册给建议
5. 危险操作要人工批

一句话：SOP = 把老师傅的经验写成「照着做就不会漏」的步骤清单；Skill 把这个清单打包给 Agent 用。
