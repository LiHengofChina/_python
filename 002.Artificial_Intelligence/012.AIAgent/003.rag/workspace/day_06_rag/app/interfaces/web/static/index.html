运维 Agent Demo

当前版本能力一览（按学习路径累积）
  • 01
    Chat · 基础对话
    FastAPI + LangChain，接入本地 Ollama（qwen2.5:7b），完成最基础的问答闭环。
  • 02
    会话管理
    多会话、消息落库（SQLite），支持多轮上下文。
  • 03
    工具调用（Tool Calling）
    模型按需调用只读运维工具（如磁盘、登录用户），用现场数据回答。
  • 04
    连接管理
    统一维护 Linux 等连接档案；Tool 按 label 解析凭证，配置与对话解耦。
  • 05
    SSE 流式过程
    聊天走 Server-Sent Events，实时展示 Thinking / Tool / Answer（可折叠）。
  • 06
    文档管理（知识库）
    上传 / 删除 / 列表；同步索引完成切块与入库，状态可见（pending / ready / failed）。
  • 07
    RAG 检索增强
    提问前 Chroma 检索，将参考资料注入 Prompt，再调用 LLM / Tool 生成答案。
栈：FastAPI · LangChain · Ollama · Chroma · SSE · 端口 8101
