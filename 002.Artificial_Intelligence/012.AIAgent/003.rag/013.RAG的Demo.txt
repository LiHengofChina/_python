最小 Prompt 模板（核心）

你是运维助手。请根据以下资料回答问题，资料中没有的请说「不知道」。

【参考资料】
{检索到的块1}
{检索到的块2}
{检索到的块3}

【用户问题】
{用户输入}

【回答】
「资料中没有就说不知道」 → 减少胡编，很重要。

//======================================================== Python 最小 Demo 逻辑 
//======================================================== Python 最小 Demo 逻辑 

# 1. 加载文档
docs = load("运维手册.txt")

# 2. 切块
chunks = split(docs, chunk_size=500, overlap=50)

# 3. Embedding + 存入向量库
vector_db = Chroma()
vector_db.add(chunks)  # 内部会调 Embedding API

# 4. 用户提问
question = "磁盘满了怎么处理？"

# 5. 检索
related = vector_db.search(question, top_k=3)

# 6. 拼 Prompt，调大模型
prompt = f"参考资料：{related}\n问题：{question}"
answer = llm.chat(prompt)

print(answer)
