（1）Tools		可执行的操作					//ssh_exec、send_email
（2）Resources	只读数据						//运维手册、配置清单
（3）Prompts		预置 Prompt 模板				//「故障排查模板」

Agent 客户端连上 MCP Server 后，自动发现有哪些 Tool/Resource 可用。

 Cursor IDE    自研 Agent 前端
