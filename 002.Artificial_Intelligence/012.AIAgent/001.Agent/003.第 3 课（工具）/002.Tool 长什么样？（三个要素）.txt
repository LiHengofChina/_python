

//=================================== 每个 Tool 需要定义：
//=================================== 每个 Tool 需要定义：


（1）name		 ： 工具名，模型用来选 		  //例子 ssh_exec
（1）description	 ： 干什么、何时用（给模型看） //「在远程主机执行 shell 命令，用于查磁盘、进程等」
（2）parameters   : 参数 schema（JSON）	  //host, command


//=================================== 示例定义（概念 JSON）
//=================================== 示例定义（概念 JSON）
{
  "name": "ssh_exec",
  "description": "在已配置的主机上执行 shell 命令，用于运维排查",
  "parameters": {
    "type": "object",
    "properties": {
      "host": { "type": "string", "description": "目标主机标识" },
      "command": { "type": "string", "description": "要执行的命令，如 df -h" }
    },
    "required": ["host", "command"]
  }
}
description 写得好不好，直接影响模型会不会用、用对不对。
