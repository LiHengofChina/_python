//=================================== Tool 是什么？

Tool = 大模型可以调用的外部函数/API

LLM 自己只能「生成文字」，
不能直接 """ SSH、查库、读文件 """ 。


"框架" 把能力封装成 Tool，模型在 """ ReAct 循环 """ 里决定 """ 调哪个工具、传什么参数 """。


//===========
//===========

LLM 输出：我要调用 ssh_exec，参数 host=XXX, command=df -h
    ↓
框架解析 → 执行真实 SSH
    ↓
结果作为 Observation 塞回给 LLM

/**
这就是 Function Calling（函数调用）
——OpenAI、Spring AI、LangChain4j 都叫这个名字或类似概念。
*/
