	┌──────────┐
	│   用户    │  输入目标 + 连接信息
	└────┬─────┘
		 ↓
	┌──────────┐
	│  Agent   │  协调中心（循环控制）
	│  编排层   │
	└────┬─────┘
		 ↓
	┌──────────┐     ┌─────────────┐
	│ Planner  │────→│   Memory    │  短期：本轮步骤/结果
	│  规划器   │     │   记忆       │  长期：历史案例/偏好
	└────┬─────┘     └─────────────┘
		 ↓
	┌──────────┐
	│   LLM    │  大脑：Thought、选 Tool、生成 Final Answer
	└────┬─────┘
		 ↓
	┌──────────┐
	│  Tools   │  手和脚
	│          │  ├─ SSH 执行命令
	│          │  ├─ RAG 查手册
	│          │  ├─ 监控 API
	│          │  └─ 告警通知
	└────┬─────┘
		 ↓ Observation 回传给 LLM
		 ↓（循环直到完成）
	┌──────────┐
	│  结果输出 │  报告 + 建议（危险操作等人审批）
	└──────────┘
