//============================================= MCP 方式（给 Agent / Cursor 用）
//============================================= MCP 方式（给 Agent / Cursor 用）

（1）MCP Server 暴露 Tool：

	Tool 名：queryDiskUsage
	参数：hostId = "linux-231"
	描述：查询 Linux 主机磁盘使用情况（df -h）

（2）Agent 不会事先知道 URL，而是：

	连上 MCP Server
	自动拿到 Tool 列表
	用户问「231 磁盘满了没？」→ 模型自己决定调 queryDiskUsage
	"MCP 返回" 给 "LLM " 看的文本结果
