
语言： Python
优势： 多个 Agent 分工协作（研究员、执行者、审核员），适合复杂任务拆分。
劣势： Python；多 Agent 成本高、难调试。

//====================================================================
//====================================================================

from crewai import Agent, Task, Crew

ops = Agent(role="运维", goal="排查故障", backstory="银行 SRE")
task = Task(description="分析 231 磁盘告警", agent=ops)
crew = Crew(agents=[ops], tasks=[task])
print(crew.kickoff())

//====================================================================
//====================================================================

vs Spring AI：

CrewAI = 多角色、多 Agent 编排（你 day_05 是单链路 Java 写死）
Spring AI 也能做多步，但多 Agent「角色团队」不是主打；要自己设计或接别的方案
