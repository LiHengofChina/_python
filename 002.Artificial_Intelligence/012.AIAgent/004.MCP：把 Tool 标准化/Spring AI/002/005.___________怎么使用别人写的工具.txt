开源 Tool 一般不是「拷进你的 MCP Server 代码里」，而是「再连一个现成的 MCP Server」。

//============================================= 要用开源工具，是 多加一条连接：
//============================================= 要用开源工具，是 多加一条连接：

你的 Agent (8105)  ──MCP──►  ops-mcp-server (8104)     ← 你的 SSH Tool
                 └──MCP──►  开源 filesystem MCP         ← 别人写的
                 └──MCP──►  开源 postgres MCP           ← 别人写的

//============================================= 【关键来了】
//============================================= 【关键来了】

Agent 启动时会从 所有已连的 MCP Server 拉 Tool 列表，模型一起用。
