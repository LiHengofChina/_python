							Location（虚拟环境存放位置）
								是你想要为虚拟环境指定的目录路径，
								也就是虚拟环境的存储位置。
								虚拟环境中的所有文件和依赖都会存储在该目录下。


"虚拟环境" 
虚拟环境是一个独立的 Python 环境，
能够在不影响系统全局 Python 设置的情况下，
管理特定项目的依赖库和解释器。
它是 Python 开发中的一个常用概念，尤其在处理多个项目时，虚拟环境可以帮助你为每个项目创建隔离的运行环境，
从而避免不同项目之间的依赖冲突。
