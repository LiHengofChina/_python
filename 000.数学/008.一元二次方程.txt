//====================================== 一元二次

为什么说：y = ax^2 + bx + c 是一元二次方程

一元二次方程是指只有一个变量的二次方程。

在给定的方程 y = ax^2 + bx + c 中，只有一个变量 x 出现在方程中，
而 y 是变量 x 的函数。

因此，这个方程被称为一元二次方程。
其中，a、b、c 是常数，它们用来确定 "方程的形状和位置" 。
方程中的 x 的 "最高次数是 2"  ，因此这是一个二次方程。


//====================================== 有一元3次方程，一元4次方程吗
//====================================== 有一元3次方程，一元4次方程吗

是的，除了一元二次方程，还有一元三次方程和一元四次方程。

一元三次方程的一般形式为:
	y = ax^3 + bx^2 + cx + d，
	//其中 a、b、c、d 是常数，且 a ≠ 0。
	//这个方程中的变量 x 的最高次数是 3。

一元四次方程的一般形式为 y = ax^4 + bx^3 + cx^2 + dx + e，
	//其中 a、b、c、d、e 是常数，
	//且 a ≠ 0。这个方程中的变量 x 的最高次数是 4。

这些方程的解法比一元二次方程更加复杂，
	//并且可能需要使用更高级的数学技巧和方法来求解。
