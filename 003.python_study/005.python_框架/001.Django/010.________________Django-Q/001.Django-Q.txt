//======================================================================（1）Django-Q

Django-Q 是基于 Django 的一个第三方 "任务队列和异步任务管理" 框架，
用于处理耗时任务、定时任务和分布式任务。


它依赖 Django 的 ORM 和设置，与 Django 项目深度集成，但它不是 Django 的官方组件，
而是一个独立的扩展工具。
