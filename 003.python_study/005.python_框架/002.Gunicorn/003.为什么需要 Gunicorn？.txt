（1）WSGI 标准：Gunicorn 是实现 WSGI 的服务器，
	 而大部分 Python Web 框架都支持 WSGI 协议。

（2）高效的进程管理：Gunicorn 可以运行 "多个进程" 以处理 "并发请求"。

（3）与 Web 服务器的配合：
		Gunicorn 负责 "Python 应用" 的运行，
		Nginx 负责反向代理和静态资源的处理，各司其职。
