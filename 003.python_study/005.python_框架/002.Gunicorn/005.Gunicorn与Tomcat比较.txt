Gunicorn 类似于 Java 中的 Servlet 容器（如 Tomcat），负责运行 Web 应用并处理请求，

而 Django 类似于 Spring MVC，是一个 Web 开发框架，专注于业务逻辑的实现。


Gunicorn ≈ Servlet 容器（如 Tomcat）
Django ≈ Spring MVC
