（1）Myers diff 的 "网格走对角线"
	 这是在二维网格里找路径，本质是 序列比对 / 编辑距离问题，核心是 "组合优化"。

（2）
	 线性回归 / 岭回归 / Lasso / 多项式回归：
	 这些是 统计建模 / 机器学习方法，目标是用函数去 "拟合数据" 分布，本质是 数值预测。

//======================================
//======================================

所以它们看起来都涉及“斜线”“拟合”，但性质完全不同：
	diff 算法：确定性 → 找最短路径
	回归算法：概率/统计 → 拟合最优函数

换句话说：在 diff 里用回归（包括多项式回归）是没有意义的，它不是 "预测" 问题，
而是 严格的序列对比问题。
