//============================================== #脚本执行方式
单独执行一个脚本：sh 脚本 ARG1 ARG2 ARG3

调用main脚本执行所有子脚本：sh main.sh ARG1 ARG2 ARG3

#参数取值
ARG1：输入true或false，true表示所有子脚本执行结果输出到文件中，
	  false表示子脚本执行结果输出到控制台
ARG2：输入Oracle数据库SID(实例名)
ARG3：输入IP地址
