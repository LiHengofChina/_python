
https://archerydms.com/


Archery 是由中国的开发者团队创建的一个开源项目。
它是一个面向数据库 SQL 审核和查询分析的平台，
旨在提高数据库管理和开发的效率。
最初由 「维他命 DBA 团队」（Vitamin DBA Team）主导开发，
后来在开源社区中得到了广泛的支持和贡献。


Archery 项目最初是 archer 的一个分支项目，
旨在提升 DBA 的工作效率，
支持多种数据库的 SQL 上线和查询功能。 
 

根据 GitHub 上的提交记录，
Archery 项目在 2018 年左右开始活跃开发。 
 

因此，Archery 项目大约于 2018 年由中国的开发者团队开始开发。
