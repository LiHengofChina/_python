

Swagger 是一个用于 "描述" 和 "文档化"  RESTful API 的工具集。
它通过一个开放标准（OpenAPI 规范）来定义 API 的结构，
		包括请求方法、路径、参数、响应格式等信息。

//====================================================== Swagger 提供了以下功能：
//====================================================== Swagger 提供了以下功能：

（1）API 文档自动生成：
		 通过注解或者 YAML/JSON 文件定义 API，
		 Swagger 可以自动生成交互式文档。

（2）交互式界面：
		 Swagger UI 提供一个用户界面，
		 允许开发者直接在 "文档中" "进行 API 调用"，方便调试和测试。

（3）客户端生成：基于 API 描述生成不同语言的客户端代码，
		简化开发工作。

Swagger 主要帮助开发团队 "快速了解" 和 "测试 API 接口"，
		增强文档的 "可读性和易用性"。
