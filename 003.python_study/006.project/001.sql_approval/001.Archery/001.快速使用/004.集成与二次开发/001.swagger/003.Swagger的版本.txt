
//====================================================== Swagger 的版本
//====================================================== Swagger 的版本

（1）Swagger 2.0：
	这是 Swagger 的一个早期版本，
	支持 YAML 和 JSON 格式的 API 描述。
	它已经被广泛使用，
	但不再积极更新，新的开发更多是围绕 OpenAPI 3.0 进行。



（2）OpenAPI 3.0：
	这是 Swagger 2.0 的继任者，
	提供了更多的功能和改进。
	例如，它增强了对请求和响应模型的支持，
	更灵活的参数定义，
	以及更清晰的描述安全性、认证方式等。
	OpenAPI 3.0 已成为行业标准。

（3）Swagger 3.x：
	虽然 OpenAPI 3.0 被广泛使用，
	但 Swagger 也在持续更新。Swagger 3.x 主要是对 OpenAPI 3.0 
	进行了一些工具和实现方面的改进，
	如 Swagger UI 和 Swagger Editor
