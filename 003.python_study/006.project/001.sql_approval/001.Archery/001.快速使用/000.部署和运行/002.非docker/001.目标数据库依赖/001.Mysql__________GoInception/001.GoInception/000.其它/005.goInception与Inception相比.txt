GoInception 在 Inception 的基础上做了一些重要扩展



（1）性能优化：GoInception 使用 Go 语言重写，相比原本的 Java 实现，
			   性能得到了显著提升，尤其在处理大量 SQL 审核时更高效。

（2）支持更多数据库：GoInception 增加了对 MySQL、MariaDB、TiDB 等数据库的支持。

（3）功能扩展：
	（3.1） 增强了 SQL 语句优化 的能力，包括更多的规则和检测项。
	（3.2） 提供了更灵活的 API 接口，方便集成到其他系统或平台。

（4）并发支持：由于 Go 语言的并发特性，
	  GoInception 可以更好地支持大规模并发的 SQL 审核任务。

（5）配置和使用简化：GoInception 在配置和使用方面做了简化，更易于部署和集成。
