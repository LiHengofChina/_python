//=================================================== Goinception 的核心功能

（1）SQL 审核：
	自动检查 SQL 语句是否符合规范（如表结构设计、索引使用等）。
	帮助检测潜在的风险操作，例如大表更新、全表扫描等。


（2）SQL 执行：
	支持分批执行 SQL（如大数据量更新），以降低数据库压力。
	支持 SQL 回滚功能，用于快速恢复误操作。

（3）SQL 优化建议：
	提供优化建议，例如索引推荐、执行计划分析等。

（4）支持多种数据库：
	兼容 MySQL 等主流数据库。
