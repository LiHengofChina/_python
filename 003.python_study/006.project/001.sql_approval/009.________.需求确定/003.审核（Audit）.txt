//========================================================================  审核（Audit）


当你点击 "SQL检测" 按钮时，实际上是在执行 "对 SQL 语句的审核"，
检查 SQL 是否符合 "一定的规范" 或者是否 "存在潜在的风险问题"。

这个过程通常包括：

（1）SQL 语法检查：确保 SQL 语法没有错误。

（2）SQL 安全性检查：检查 SQL 是否存在潜在的 
					 SQL 注入漏洞、危险操作（例如删除、更新大量数据）等。
（3）执行效率检测：检查 SQL 是否存在性能问题（如全表扫描、大量 JOIN 操作等）。

（4）数据库兼容性检测：确保 SQL 在目标数据库中能够正确执行，不会导致问题。
