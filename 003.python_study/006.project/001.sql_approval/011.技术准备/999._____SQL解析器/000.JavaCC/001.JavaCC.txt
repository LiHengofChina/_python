JavaCC（Java Compiler Compiler）是一个 "开源的"、"Java 语言编写的"编译器生成器。

它可以用来 "定义语言的语法（语法规则）" 并自动生成 "解析器"。

简而言之，JavaCC 是一个工具，
可以 "帮助开发者" 创建能够 "解析自定义语言（包括 SQL 语句）" 的 "程序"。

//=========================================================== JavaCC 的关键特点：
//=========================================================== JavaCC 的关键特点：

（1）语法分析：
		你通过定义语法规则（例如，BNF 或 EBNF 语法）告诉 JavaCC 
		"如何解析" 输入的 "文本数据"。

（2）生成解析器：
		JavaCC 根据 "定义的语法" 自动生成 "Java 代码"，构建一个解析器，

		可以将 "输入的文本（例如 SQL 语句）" 转化为 "结构化的数据（如抽象语法树，AST）"。

（3）灵活性：JavaCC 能够处理 "复杂的语法规则" 和 "语言" 。


//=========================================================== 在 SQL 解析中的应用：
//=========================================================== 在 SQL 解析中的应用：

JSQLParser 和 Apache Calcite 都使用 JavaCC 来解析 SQL 语句。
这意味着它们会根据 "SQL 语法规则" 生成 Java 代码，
并用 "生成的代码" 来分析 "SQL 查询" 并构建 "抽象语法树（AST）"。


简言之，JavaCC 是帮助 "解析" 和 "处理自定义语言（如 SQL）" 的工具，
它使得开发 " SQL 解析器" 变得更加 "高效和结构化"。
