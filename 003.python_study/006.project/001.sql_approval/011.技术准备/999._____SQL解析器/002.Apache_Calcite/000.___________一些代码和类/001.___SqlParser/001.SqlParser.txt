
//================================== SqlParser 类
//================================== SqlParser 类

SqlParser 类 SqlParser 是 Apache Calcite 提供的 "SQL 解析器"，
用于将 SQL 语句解析成 "SqlNode 语法树"。


//================================== 作用
//================================== 作用

//=== 解析 SQL 语句
//=== 解析 SQL 语句
（1）将 SQL 转换成 SqlNode 对象（AST，抽象语法树）
（2）供后续 SQL 校验、优化、执行 使用

//=== 支持自定义解析配置
//=== 支持自定义解析配置
（1）设置 SQL 语法规范（SqlConformance）
（2）控制 大小写转换、标识符解析 规则
（3）自定义 SQL 解析模式
