

//================================ Prepare 类简介
//================================ Prepare 类简介


Prepare 是 Apache Calcite 里的一个核心抽象类，
用于SQL "解析、验证、优化、执行计划" 生成。


//================================ 主要作用
//================================ 主要作用


（1）SQL 转换 ：Prepare 负责把 SQL 语句 转换为 Calcite 的内部表示（SqlNode → RelNode）。

（2）SQL 验证
		语法验证（类似 SqlValidator）
		类型检查（确保数据类型匹配）
		权限检查（如果有）

（3）优化 SQL
		应用 Calcite 的查询重写、优化规则。

（4）生成执行计划
		生成 RelNode 结构，提供给 SQL 执行引擎使用。


//================================ Prepare 的常见子类
//================================ Prepare 的常见子类

CalcitePreparingStmt（核心实现）
	//（1）继承 Prepare，负责 SQL 解析、验证、转换为执行计划。

CalciteMaterializer
	//（2）继承 CalcitePreparingStmt，用于物化查询。

//================================ 什么时候用 Prepare？
//================================ 什么时候用 Prepare？

（1）需要执行计划（RelNode）
（2）优化 SQL（查询重写、索引优化等）
（3）查询数据库 schema、数据类型检查

如果只是解析 SQL 并进行 SQL 审核，用 SqlParser 更轻量级。
