HepPlanner 和 VolcanoPlanner 都是 Calcite 提供的查询优化器：

（1）HepPlanner (基于规则的优化器，RBO)：
	（1）使用 规则 来执行查询优化，逐步应用优化规则。
	（2）适用于规则驱动的优化，适合简单的查询。
	（3）速度较快，因为它按规则应用优化，没有复杂的代价评估。

（2）VolcanoPlanner (基于代价的优化器，CBO)：
	（1）使用 代价模型 来评估执行计划的优劣，选择最优方案。
	（2）适用于复杂查询和需要考虑多个优化策略的场景。
	（3）速度较慢，因为它需要评估更多的执行计划和代价模型。

	总结：
		HepPlanner 适用于简单的查询，优化较快。
		VolcanoPlanner 适用于复杂查询，优化较精确但较慢。
