//===================================== 动态规划在VolcanoPlanner 里面起什么作用


在 VolcanoPlanner 中，动态规划用于生成和评估不同的查询执行计划。
它通过递归地拆分查询操作（如连接、过滤、聚合等），并使用 代价模型 来选择最优的执行计划。

动态规划帮助 VolcanoPlanner 在考虑所有可能的执行计划时，
能够高效地选择代价最小的方案，从而优化查询性能。


//===================================== 在优化时，它会连接到数据库吗？
//===================================== 在优化时，它会连接到数据库吗？

在优化时，VolcanoPlanner 不会连接到数据库。
它只会基于 查询的结构 和 代价模型 来生成优化的执行计划，而不涉及实际的数据访问。
数据库连接通常发生在 执行阶段，即优化后的查询计划被提交到数据库执行时。
