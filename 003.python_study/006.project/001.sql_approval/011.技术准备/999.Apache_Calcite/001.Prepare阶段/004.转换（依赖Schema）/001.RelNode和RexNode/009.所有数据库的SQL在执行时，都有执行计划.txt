//=============================================== 所有数据库的SQL在执行时，都有执行计划


所有数据库在执行 SQL 时都有执行计划。

有的数据库 公开（如 MySQL EXPLAIN、Oracle 执行计划、PG 计划树）
有的数据库 不公开内部细节
有的数据库执行计划 比较简单
SQL 再简单、数据库再弱，也必须先生成一个 执行策略 才能执行

任何数据库要执行 SQL，都必须有执行计划，只是复杂程度不同。
