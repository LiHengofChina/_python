可以用 Apache Calcite 来执行 SQL，但要分两种情况。

//=====================================
//=====================================
1. Calcite 能不能“执行 SQL”？能，但方式不同于 JDBC

	一个 SQL 解析 + 优化 + 查询规划框架，不是数据库。

它没有存储、没有执行引擎。

所以 Calcite 有两种执行方式：
	方式 A：Calcite 作为“代理查询器”去执行 SQL（常见方式）
	方式 B：Calcite 自己执行 SQL（基于 Schema + Interpreter）
	Calcite 可以执行 SQL，但没必要。你的场景用 Calcite 分析，用 JDBC 执行，是最佳方案。
