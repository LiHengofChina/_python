

Elasticsearch 是一个分布式的全文搜索和分析引擎，基于 Apache Lucene 构建。
它可以快速存储、搜索和分析大量数据，常用于 "日志分析"、"全文搜索" 和 "实时数据" 处理。

//========================================== 核心特点：
//========================================== 核心特点：

（1）全文搜索： 支持快速、精准的全文搜索。
（2）实时性： 数据写入后几乎立即可被搜索。
（3）分布式： 天生支持分布式架构，能处理海量数据。
（4）RESTful API： 使用简单的 HTTP 接口进行交互。
（5）强大的聚合分析： 支持复杂的数据分析，比如统计、分组等。

//========================================== 常见用途：
//========================================== 常见用途：

日志和监控： 与 Logstash、Kibana 一起构成 ELK 堆栈，用于日志收集、存储和展示。
全文搜索： 网站、应用的全文检索功能（如商品搜索）。
数据分析： 对大量数据进行实时聚合分析。
总结： Elasticsearch 是用来搜索和分析数据的强大工具，尤其适合处理海量数据。
