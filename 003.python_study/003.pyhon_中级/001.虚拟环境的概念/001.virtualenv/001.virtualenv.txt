//============================================ virtualenv 是一个工具

它的主要目的是为 Python 项目创建独立的虚拟环境。
	 这些虚拟环境可以 "隔离项目所需的依赖包"，与 "系统的全局 Python 环境分开"，避免依赖冲突。


工具的功能：
	为项目创建隔离的运行环境。
	在每个环境中可以有独立的依赖库。
	支持多个不同的 Python 版本和环境。
