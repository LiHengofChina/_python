//===================================================== 镜像（Image）
在 Docker 中，镜像（Image） 是一个包含了应用程序及其
	所有 "依赖、库、配置" 和 "文件系统" 的只读模板。
	它是容器的 "蓝图" ，包含了 "运行应用" 所需的一切。

例如，Archery 项目的 Docker 镜像可能包含了 Archery 的所有源代码、
	运行时环境、依赖项等。
简单来说，Docker 镜像是你运行 Docker 容器所依赖的基础，它定义了容器的环境。
