
//============================================= 示例：
//============================================= 示例：

	docker exec -ti archery /bin/bash


（1）docker exec  表示在运行的窗口内部执行
		//容器必须是正在运行的，才能使用 docker exec。

（2）-ti
		-t：分配一个伪终端（TTY），方便在容器中运行交互式命令。
		-i：保持容器的标准输入（stdin）打开，用于接收用户的输入。

（3）archery
		指定要进入的容器的 "名字或容器" ID。
		这里指的是名为 archery 的容器。
