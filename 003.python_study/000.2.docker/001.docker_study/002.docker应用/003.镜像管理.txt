

注意是：镜像管理，不是docker容器管理

//======================================================================= （4）镜像
//======================================================================= （4）镜像


（1）通过命令查看镜像列表
	 docker images


（2）查看某个镜像的详细
	 docker inspect hello-world


（3）删除镜像
	 docker rmi hello-world		//删除镜像
	 docker rmi -f hello-world	//强制删除镜像
