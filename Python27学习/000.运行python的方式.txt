//====================================运行python代码的方式
1). 交互式编程
	不需要创建脚本文件，
	是通过 "Python 解释器"的交互模式来编写代码。
	
2). 使用py命令来执行脚本 python test.py//常用这种
	Python 脚本文件将以 .py 为扩展名
	
3). 给脚本添加执行权限来执行脚本
	==========在脚本内容中指定"Python 解释器"
	#!/usr/bin/python
	print "Hello, Python!";
	==========
	$ chmod +x test.py
	$ ./test.py
